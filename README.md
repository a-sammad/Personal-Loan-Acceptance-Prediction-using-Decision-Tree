# Personal-Loan-Acceptance-Prediction-using-Decision-Tree
project focuses on predicting whether a bank customer will accept a personal loan offer using a Machine Learning classification model. The model is trained using the Decision Tree Classifier from Scikit-Learn.  The goal of this project is to analyze customer demographic and financial information and determine the likelihood of loan acceptance.
Project Overview

Banks often run marketing campaigns to offer personal loans to customers. However, not every customer accepts the offer. By applying machine learning techniques, we can predict which customers are more likely to accept the loan, helping banks improve marketing efficiency and decision-making.

This project includes:

Data loading and preprocessing

Exploratory Data Analysis (EDA)

Feature encoding

Model training using Decision Tree

Model evaluation using accuracy and confusion matrix

Technologies Used

Python

Pandas

Matplotlib

Seaborn

Scikit-Learn

Exploratory Data Analysis

Several visualizations were created to understand the dataset:

Age Distribution using histogram

Job vs Loan Acceptance using count plot

Marital Status vs Loan Acceptance

These visualizations help identify patterns and relationships between customer attributes and loan acceptance.

Machine Learning Model

The project uses a Decision Tree Classifier to perform binary classification.

Steps performed:

Dataset loaded using Pandas

Target variable separated from features

Categorical variables converted using One-Hot Encoding

Dataset split into training (80%) and testing (20%)

Model trained using Decision Tree

Predictions generated on test data

Model evaluated using:

Accuracy Score

Confusion Matrix

Model Evaluation

The model performance is measured using:

Accuracy Score – measures prediction correctness

Confusion Matrix – shows true positives, false positives, true negatives, and false negatives

Project Purpose

The main purpose of this project is to demonstrate how machine learning models can assist financial institutions in predicting customer behavior and improving marketing strategies.
